1. Método sort()

Explicação: O método sort() ordena os elementos de um array. Por padrão, ele converte os elementos para strings e os ordena de forma crescente ou alfabetica. Este metodo pode alterar o array original


const numerosOrdenados = [...numeros].sort((a, b) => a - b);

2. Método filter()

Explicação: O método filter() cria um novo array contendo todos os elementos que passam em um teste fornecido por uma função. No exemplo abaixo, estamos filtrando números maiores que 3.


const numerosMaioresQue3 = numeros.filter(num => num > 3);

3. Método map()

Explicação: O método map() cria um novo array a partir de um ja existente com aplicando uma função em cada elemento do array original. Aqui, estamos calculando o quadrado de cada número no array.


const quadrados = numeros.map(num => num * num);

4. Método reduce()

Explicação: O método reduce() aplica uma função a um array e/ou a cada valor do array (da esquerda para a direita) para reduzi-lo a um único valor. No exemplo abaixo, estamos somando todos os números do array.


const soma = numeros.reduce((acumulador, valorAtual) => acumulador + valorAtual, 0);

5. Operador Spread (...)

Explicação: O operador spread (...) permite expandir um array em seus elementos individuais, o que é útil para copiar arrays ou combinar arrays com novos elementos. No exemplo abaixo, estamos criando um novo array adicionando dois novos números ao array original.


const novosNumeros = [...numeros, 10, 11];
